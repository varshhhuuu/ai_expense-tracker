# 💰 AI Expense Tracker

## 📌 Overview
AI Expense Tracker is an intelligent personal finance management application that helps users track, categorize, and analyze their expenses efficiently. The system provides insights into spending patterns through interactive dashboards and visualizations.

---

## ✨ Features
- Add and manage daily expenses
- Automatic expense categorization
- Monthly and yearly spending analysis
- Interactive charts and dashboards
- CSV file upload support
- User-friendly interface

---

## 🛠️ Technologies Used

- Streamlit

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Expense-Tracker.git
```

### Navigate to Project Folder

```bash
cd AI-Expense-Tracker
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
